**Архетип:** Метаболические | **Локомоция:** Горизонтальная

Циклическое движение: ноги → спина → руки.

## Исходное положение

Сидя на гребном тренажёре, ноги согнуты, руки прямые, хват на рукоятке.

## Выполнение

1. Оттолкнитесь ногами, сохраняя корпус прямым
2. Отклоните корпус назад до угла 45°
3. Подтяните рукоятку к груди, сводя лопатки
4. Вернитесь в исходное положение в обратном порядке: руки → корпус → ноги

## Частые ошибки

- **Скругление спины** — держите спину прямой
- **Раннее сгибание рук** — сначала ноги, потом руки
- **Неполная амплитуда** — полностью выпрямляйте ноги в фазе тяги
