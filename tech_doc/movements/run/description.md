**Архетип:** Метаболические | **Локомоция:** Горизонтальная

Циклическая локомоция с фазой полёта.

## Исходное положение

Стоя, ноги на ширине таза, корпус слегка наклонён вперёд, руки согнуты в локтях.

## Выполнение

1. Начните движение с отталкивания стопой, выбрасывая бедро вперёд-вверх
2. Приземляйтесь на переднюю часть стопы с последующим перекатом
3. Руки работают разноимённо с ногами
4. Корпус сохраняет лёгкий наклон вперёд

## Частые ошибки

- **Приземление на пятку** — приземляйтесь на переднюю часть стопы
- **Наклон корпуса назад** — держите лёгкий наклон вперёд
- **Скрещивание рук перед грудью** — руки движутся вдоль корпуса
